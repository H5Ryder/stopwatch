# React + TypeScript + Vite Stopwatch

Live Demo: https://stopwatch-iy2r.vercel.app/

To run use the following commands:

```
npm install
npm run dev
```

Functionality:
- Start/Stop
- Lap
- Reset
- Save image of recorded laptimes
